# Quiz App

## Overview

Quiz App is a simple command-line quiz application built using Python. The application asks users a series of technology-related questions, evaluates their answers, calculates the final score, and displays the percentage obtained.

This project is designed for beginners who want to practice Python fundamentals such as lists, dictionaries, loops, conditional statements, user input, and basic program flow.

---

## Features

* 10 technology-related quiz questions
* User-friendly command-line interface
* Instant feedback for each answer
* Score calculation
* Percentage calculation
* Case-insensitive answer checking

## How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd Quiz-App
```

3. Run the application:

```bash
python app.py
```
